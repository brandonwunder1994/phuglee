Version RMW:
```js
function saveBrain(brain, { expectedVersion } = {}) {
  const current = loadBrain();
  if (expectedVersion != null && current.version !== expectedVersion) {
    const err = new Error('Brain version conflict');
    err.code = 'VERSION_CONFLICT';
    err.statusCode = 409;
    throw err;
  }
  // enforceBrainCaps; version = (current.version||0)+1; metrics = recomputeMetrics; atomic write
}
```

API:
```http
POST /api/bridge/brain/undo
Headers: x-phuglee-user: admin
Body: { "brainVersion"?: number }
→ 200 { ok, brainSummary }
→ 400 { code: 'NOTHING_TO_UNDO' }
→ 409 { code: 'VERSION_CONFLICT', currentVersion }
→ 403 ADMIN_REQUIRED

GET /api/bridge/brain/metrics
→ 200 {
  totalDecisions,
  typeRulesActive, phraseRulesActive, phraseRulesProposed,
  suppressCount, promoteCount
}
```
