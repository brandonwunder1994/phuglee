    3. **requireAdmin(req):**
       ```js
       function requireAdmin(req) {
         const username = readPhugleeUser(req);
         if (username !== ADMIN_USERNAME) {
           const err = new Error('Admin required');
           err.code = 'ADMIN_REQUIRED';
           err.statusCode = 403;
           throw err;
         }
         return username;
       }
       ```
       Strict even if AUTH_DISABLED — do not open brain writes.

    4. **handleBrainDecision(req, res):**
       - try requireAdmin → on ADMIN_REQUIRED sendJson 403 `{ error, code: 'ADMIN_REQUIRED' }`
       - readBody; if length > MAX → 413 PAYLOAD_TOO_LARGE
       - JSON.parse → catch 400 INVALID_JSON
       - Validate action ∈ approve|deny, section ∈ distressed|not_distressed, Array.isArray(rowIds); else 400 MISSING_FIELDS or INVALID_DECISION
       - If uploadType === 'water_shut_off' → 400 WATER_TRAINING_UNSUPPORTED
       - Optional: if (deny distressed or approve FN) and zero rowIds present in target list → 400 ROW_IDS_NOT_FOUND
       - `const brain = loadBrain()`
       - `const result = applyDecision(body, { brain, currentRows: body.rows, notDistressedRows: body.notDistressedRows, by: username })`
       - Catch INVALID_DECISION → 400
       - `saveBrain(result.brain)`
       - sendJson 200:
         ```js
         {
           ok: true,
           rows: result.rows,
           notDistressedRows: result.notDistressedRows,
           reviewGroups: result.reviewGroups,
           event: result.event,
           brainSummary: result.brainSummary,
           statsPatch: { kept: result.rows.length, notDistressed: result.notDistressedRows.length }
         }
         ```
