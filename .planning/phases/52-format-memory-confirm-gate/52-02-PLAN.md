**Fingerprint algorithm (prescriptive):**
```js
// 1. map normalizeHeader
// 2. filter empty and '_meta'
// 3. sort (order-independent)
// 4. join '\u0001'
// 5. crypto.createHash('sha1').update(base, 'utf8').digest('hex')
// Do NOT hash row counts or file bytes
```
