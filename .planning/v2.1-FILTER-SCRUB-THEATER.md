## Style DO / DON’T

### DO
- Asymmetric desk (dominant + scrap) like Collect
- Solid cream Anton H1 left-aligned
- Ops voice: Scrub the Mess · Drop the clerk file · Stage the list
- Ember/gold heat for success and kill/kept hierarchy
- Territory/Command HUD for big numbers
- `phuglee-btn` primary vocabulary
- `prefers-reduced-motion` for all theater

### DON’T
- Equal 3-card feature rails
- Centered gradient marketing H1
- Green SaaS success flashes
- Peer Train/Kept/Brain tabs when open groups exist
- Auto-push Analyze
- Wipe lists/brain data
- New framework
