# v1.2 Premium Brand Experience — Design Specification

**Milestone:** v1.2  
**Reference:** Login page (`public/index.html`, `landing.css`, `auth.css`)  
**Logo reference:** `public/images/phuglee-logo.svg`  
**Date:** 2026-07-06  
**Status:** Approved

---

## Problem

The login page delivers a cinematic, premium, badass first impression — distressed home photo, film grain, wear scratches, huge Phuglee mascot, ember CTAs. After sign-in, pages fall back to flat brown gradients and generic Heat grid. The product feels like two different brands.

## Solution

Extract the login page's visual DNA into a reusable **Premium Atmosphere** layer and apply it page-by-page across every post-login surface while keeping the login page untouched.

---

## Brand DNA (from logo + login)

### Logo palette (extracted from `phuglee-logo.svg`)

| Swatch | Hex | Use |
|--------|-----|-----|
| Cream highlight | `#f4efe6` / `#f5f3ec` | Primary text, logo fills |
| Warm stone | `#aea38f` / `#a59a88` | Secondary surfaces, muted chrome |
| Charcoal | `#353630` / `#383832` | Deep shadows, outlines |
| Ember flame | `#e58435` / `#ac6b32` | Accent gradients, signal heat |
| Gold core | `#eeb746` / `#e2bb60` | CTA highlights, tier badges |

### Atmosphere stack (login page)

```
premium-atmosphere
├── premium-photo        → ugly-house.jpg, grayscale, low opacity
├── premium-photo-grain  → SVG fractalNoise tile
├── premium-photo-wear   → photo-wear.png scratches
├── premium-vignette     → radial + top/bottom letterbox darkening
└── heat-field (subtle)  → ember glow, grid hidden or ultra-faint
```

### Texture on panels (auth modal)

- Top ember gradient hairline (`auth-card-face::after`)
- Diagonal scratch lines (`repeating-linear-gradient`)
- Grain overlay at 12% opacity
- Inset highlight + deep drop shadow
- Hot border on hover (`--border-hot`)

### Typography & CTAs

- **Anton** — uppercase heroes, step numbers, pricing amounts
- **Outfit** — body, labels, descriptions
- **btn-heat** — heat-core → flame → ember gradient + shine sweep
- Text shadows on taglines over photo layers

## Page-by-page upgrade plan

### OUT OF SCOPE — Login (`/`)

**Status:** Locked — do not modify.

Reference only for: atmosphere, auth modal panels, btn-heat, logo scale treatment.

---

### Phase 15 — Shell & Global Nav

**Files:** `shell-nav.css`, `shell.css`, `shell-nav.js`, `rewrite.js` injection

| Upgrade | Detail |
|---------|--------|
| Nav chrome | Glass bar with ember hairline top (like auth card), deeper blur, subtle grain |
| Brand mark | Text logo with cream color matching SVG; hover ember glow |
| Active links | Heat-core text + ember tint — add subtle wear on active pill |
| Content offset | Ensure `premium-bg` doesn't clash with fixed nav z-index |
| Embedded modules | Injected pages get `premium-atmosphere.css` via rewrite |

**Success look:** Nav feels like it belongs on the login page — not a separate admin header.

---

### Phase 16 — Command Hub / How It Works (`/heat`)

**Files:** `heat.html`, `hub.css`, `heat-atmosphere.css`

| Section | Current | Upgrade |
|---------|---------|---------|
| Page backdrop | `heat-bg` glow + grid | Add `premium-bg` at 25–35% opacity behind content |
| Hero | Plain text on dark | Cream headline with gradient flame clip; tagline with text-shadow |
| Hotspot panel | Flat elevated card | `premium-panel` + grain + ember top hairline |
| Signal pills | Small uppercase chips | Slightly larger, wear texture on border, ember dot bullets |
| Step cards (01–03) | Number + body | Oversized Anton numbers with ember glow; panel grain borders |
| Filter box | Nested list | Auth-style inset dark field + hot border |
| Pricing grid | Hub cards | Match `auth-pricing-card` — slide-right hover, featured badge, dashed exclusive |
| CTA bar | Bottom strip | Full-width ember gradient bar with grain overlay |
| Footer | Muted line | Cream stone color, subtle top border |

**Hero copy treatment:** "Leads this hot don't wait" — flame gradient on key words like landing title.

---

### Phase 17 — Collect Records (`/collect`)

**Files:** `collect.html`, `collect-records.css`, `landing.css` (shared)

| Section | Current | Upgrade |
|---------|---------|---------|
| Page backdrop | Standard heat-field | Distressed home + subdued heat glow (collect is an action page) |
| Hero | `landing-title` gradient | Scale Anton headline; add `premium-tagline` shadows |
| CTAs | btn-heat + secondary | Shine sweep on primary; secondary gets wear border on hover |
| Start Requests dialog | Good foundation | Add `premium-panel-grain`, ember top line, auth-checkbox styling on radios |
| Choice cards | Radio tiles | Hover translateX like auth-pricing-card; selected = ember glow ring |
| PDF Filler info dialog | Basic modal | Match auth modal backdrop blur + rise animation |

**Vibe:** "Hit Up the City!" should feel like a battle cry — bold, textured, not a SaaS onboarding screen.

### Phase 18 — Data Bridge (`/bridge`)

**Files:** `bridge.html`, `bridge.css`

| Section | Current | Upgrade |
|---------|---------|---------|
| Page backdrop | Minimal radial gradient | Premium photo at very low opacity (utility page — don't overpower) |
| Hero | "Send to Analyzer" | Eyebrow + Anton title + lead with premium typography |
| Step panels | Flat cards | `premium-panel` with step number badges (1 · 2 · 3) in ember circles |
| Mapping table | Basic table | Dark inset rows, hot border on focus, grain header row |
| Preview table | Scrollable grid | Analyzer-adjacent chrome — monospace headers, ember scroll thumb |
| Actions | bridge-btn | Primary = btn-heat; ghost = wear-border secondary |
| Error alert | Red text | Auth-error panel styling (inset danger) |

**Note:** Bridge is a workflow utility — premium but restrained. Photo layer at ~15% opacity max.

| Surface | Upgrade |

### Phase 21 — Cross-App Premium QA

**Audit route:** Sign in → Hub → Collect → Bridge → each Forge page → Analyzer → back to Hub

| Check | Criteria |
|-------|----------|
| Atmosphere | Distressed layer visible but readable on every page |
| Typography | Anton headlines, Outfit body, cream text consistent |
| CTAs | All primary actions use btn-heat or hub-btn-primary gradient |
| Panels | Cards use grain + ember hairline, not flat borders |
| Nav | Same premium chrome, correct active states |
| Contrast | Body text ≥ 4.5:1 on all surfaces |
| Motion | Reduced-motion respected |
| Tests | distress-os `npm test`, Forge `gsd.py verify`, Analyzer `npm test` green |
