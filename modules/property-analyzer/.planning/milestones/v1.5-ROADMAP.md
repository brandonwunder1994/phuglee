## Goal

Trustworthy distressed-property classification with **Street View first**, **satellite only when needed**, **review when still unclear**.

---

## Phases

### Phase 21: Street-First Routing + Decision Fix

**Goal:** Fix the two biggest accuracy bugs — bad routing on blurred SV and biased prompts/rules.

**Items:** M4-01, M4-02, M4-03

**Success criteria:**
1. Street View runs first; satellite only on absent/blurred/obstructed (cache before API)
2. Prompts no longer default uncertain cases to well_maintained
3. Tier engine stops silent downgrade of score ≥6 without manicured proof
4. ≥8 routing fixture tests + updated tier fixtures
5. `npm test` passes

**Plans:** TBD — `/gsd:plan-phase 21`

---

### Phase 22: Confidence + Imagery Edge Cases

**Goal:** Route uncertain classifications to review; clean up blurred vs unavailable.

**Items:** M4-04, M4-05, M4-06

**Success criteria:**
1. `classificationConfidence` on records; thresholds documented
2. Low confidence → tier review lane (existing review UI extended)
3. Blurred/obstructed/unavailable/retry paths distinct
4. Obstructed SV triggers satellite (cached first) before review
5. `npm test` passes

**Plans:** TBD — `/gsd:plan-phase 22`

---

### Phase 23: Regression + Learning Loop

**Goal:** Safe iteration — golden set + balanced learned rules.

**Items:** M4-07, M4-08

**Success criteria:**
1. Golden-set replay script from audit JSONL + review corrections
2. Distress-promoting learned rules apply symmetrically
3. Baseline tier distribution report before/after rule change
4. `npm test` passes

**Plans:** TBD — `/gsd:plan-phase 23`

---

### Phase 24: Scale + Metrics

**Goal:** Ready for 100k leads — cache policy, re-tier without re-vision, accuracy metrics.

**Items:** M4-09, M4-10

**Success criteria:**
1. Cache-first satellite policy documented and enforced in code paths
2. `recalibratePropertyScores` documented as rule-change path (no Gemini re-call)
3. Manual classification smoke log committed
4. FN/FP rates from review corrections reported
5. Milestone closed; M4 items marked done
