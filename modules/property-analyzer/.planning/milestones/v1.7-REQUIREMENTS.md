# Requirements: Property Distress Analyzer — v1.7 Lead Export

**Defined:** 2026-07-01  
**Core Value:** Upload → scan → trust tier classifications → **export** distressed leads for dialing  
**Source:** User direction (2026-07-01) + `.planning/research/SUMMARY.md`

## v1.7 Requirements

### Full-Database Export

- [x] **EXPORT-01**: User can export the entire lead database (`state.results`) regardless of active filter or search
- [x] **EXPORT-02**: Export produces Excel (.xlsx) with a fixed dial-ready column schema (no column picker)

### Column Schema (dial-ready profile)

- [x] **EXPORT-03**: Each row includes: Cache Date, Street Address, City, State, Zip Code
- [x] **EXPORT-04**: Each row includes **Street View Image URL** — absolute cached imagery link (`/api/cached-imagery/...`); blank if not cached
- [x] **EXPORT-05**: Each row includes **Google Maps Street View Link** — browser pano URL (`google.com/maps`); works without local server; blank if no pano/location data
- [x] **EXPORT-06**: Each row includes Lead Type (Code Violation, Pre-Foreclosure, Probate, Tax Lien, Water Shut Off)
- [x] **EXPORT-07**: Each row includes Lead Category (distress tier: Distressed, Well Maintained, Vacant, etc.)
- [x] **EXPORT-08**: Each row includes Property Type (Home, Land/Lot, Blocked, Unavailable)
- [x] **EXPORT-09**: Each row includes Contact Name (first + last), Phone, Email

### Data Integrity

- [x] **EXPORT-10**: Cache Date sourced from `imagery.streetView.cachedAt` after imagery index hydration
- [x] **EXPORT-11**: Cached Street View Image URLs are absolute (clickable when local server is running)
- [x] **EXPORT-12**: Google Maps links built from `viewMeta.panoId` or `viewMeta.panoLat/panoLng`, falling back to geocoded address
- [x] **EXPORT-13**: Export row count equals `state.results.length` for database export

### Preservation

- [x] **EXPORT-14**: Existing filter-scoped detailed export preserved as `profile: 'full'` (no regression)
- [x] **QA-07**: `npm test` passes after export changes; export column mapping covered by unit tests

## Future Requirements (post v1.7)

- **EXPORT-16**: Distress Score column for Excel sort
