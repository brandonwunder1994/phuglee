## CSS Architecture Problem

```
tokens.css        → cyber palette (good, single source)
cyber-theme.css   → 604 lines, main shell + cards
cyber-ultra.css   → 646 lines, premium layer
app.css           → 6,630 lines, calm + legacy cyber + cmd-palette + review + modals
```

**Conflict:** `body.cyber-theme` rules in cyber files fight `calm-dialog`, `card-calm`, `review-mode-*`, and `cmd-palette-*` in `app.css`. Specificity wars = inconsistent surfaces.

**v1.4 strategy:**
1. **Formalize** cyber tokens in `tokens.css` (already done)
2. **Migrate** surface-by-surface from `app.css` → `cyber-theme.css` / `cyber-ultra.css`
3. **Retire** `calm-dialog`, `card-calm`, `inspector-calm` class names → `cyber-dialog`, `card-cyber`, `inspector-cyber`
4. **Delete** orphaned calm rules once migrated (target: shrink `app.css` by ~2,000 lines)
5. **Do NOT** add a 4th stylesheet — consolidate into cyber-* pair

## Premium Upgrade Playbook (per surface)

### 1. Review Mode — the crown jewel
- Full-bleed void background with subtle grid (not flat scrim)
- Cinematic image stage: neon corner brackets on street/satellite panes
- Action bar as floating HUD dock — tier-colored glow on hover, mono `<kbd>` chips
- Progress as thin cyan arc, not text-only
- Stats ("0 kept · 0 changed") in Orbitron micro-type
- Exit button = danger magenta, not generic gray

### 2. Property Inspector Modal
- Replace `inspector-calm` → cyber gauge with animated fill stroke
- Metadata panel: glass card with tier badge as hero element
- Navigation (← Prev / Next →) as HUD pills
- Kill calm placeholder SVG styling; use mono "NO SIGNAL" state

### 3. Table View
- Dark zebra rows with cyan hover line
- Tier badges match card-cyber colors exactly
- Sticky header with frosted glass
- Score column = mini neon pill (same as card-score-pill)

### 4. ⌘K Command Palette
- Backdrop: void + 40% blur (not legacy neon wash)
- Active item: left cyan bar + subtle glow (Linear-style but cyber)
- Category grouping with Orbitron section labels
- This is the "power user" layer — should feel like a terminal command interface

### 5. Modals (Settings / Upload / Brain)
- Unified `cyber-dialog` component pattern:
  - 16px radius, cyan/magenta corner accents (match panel-chrome)
  - Header: Orbitron title + mono close
  - Body: Outfit copy, JetBrains for code/diag
  - Primary CTA: cyan gradient button (reuse command bar Start Scan style)

### 6. Performance guardrails
- Card stagger: cap to first 24 visible, disable on scroll
- `backdrop-filter`: only on sticky command bar + modals, not every card
- Ambient orbs: reduce blur radius on `< 1200px` viewports
- Profile with 10k session loaded before ship
