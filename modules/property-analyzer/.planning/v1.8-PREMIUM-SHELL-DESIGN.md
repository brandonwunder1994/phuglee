# v1.8 Premium Shell — Design Document (Phase 8.1)

**Milestone:** v1.8 Premium Shell  
**Direction:** Aerial Command  
**Date:** 2026-07-04  
**Status:** Approved — user directed full execution of phases 8.1–8.6

## Problem

Distress Analyzer v1.4–v1.7 shipped a functional cyber theme, but the UI reads as generic neon decoration (Orbitron, cyan/magenta/purple, scanlines, orbs) rather than a sellable premium product. Property imagery — the core differentiator — is visually secondary to chrome.

## Chosen Direction: Aerial Command

**Metaphor:** Satellite intelligence console for distressed property investors. Photography-forward, data-confident, restrained atmosphere.

### Typography
| Role | Font | Rationale |
|------|------|-----------|
| Display | **Fraunces** (600–700) | Editorial authority; distinctive without sci-fi cliché |
| Body | **DM Sans** (400–600) | Clean, premium SaaS legibility |
| Mono | **JetBrains Mono** | Keep for HUD metrics, scan log |

**Retire:** Orbitron as primary display font.

### Color
| Token | Value | Use |
|-------|-------|-----|
| `--background` | `#0c0e12` | Deep warm void |
| `--foreground` | `#f4f1ea` | Warm paper white |
| `--accent` | `#e8a838` | Amber distress signal (primary CTA, scan energy) |
| `--accent-cool` | `#5eb3c8` | Secondary — well-maintained, system OK |
| `--tier-distressed` | `#e85d4c` | Warm coral-red |
| `--tier-well` | `#5eb3c8` | Cool teal |
| `--glass-border` | `rgba(232, 168, 56, 0.18)` | Subtle amber chrome |

**Retire:** Cyan/magenta/purple neon trifecta as dominant accents. Keep tier colors semantically distinct but warmer.

### Atmosphere
- **Remove:** ambient orbs, scanlines, heavy grain, HUD corner chrome on every panel
- **Keep:** subtle vignette, very faint grid (opacity ≤ 0.12), soft radial warmth top-center
- **Add:** imagery-forward card layout, amber scan sweep on progress bar

### Motion (respect `prefers-reduced-motion`)
- Empty state: staggered demo scan on 3 sample properties
- KPI numbers: count-up on scan complete (900ms ease-out)
- Pipeline bar: width transition on segment update
- Distressed KPI: single pulse when count > 0
- Property cards: subtle lift + amber edge glow on hover
- Review overlay: fullscreen immersion, larger imagery ratio
